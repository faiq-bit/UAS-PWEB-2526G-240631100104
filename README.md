## Deskripsi
Aplikasi web sederhana untuk mencatat keuangan pribadi (pemasukan & pengeluaran) berbasis PHP Native dan MySQL. Dilengkapi fitur CRUD, pencarian, dan perhitungan saldo otomatis.

## Teknologi
- HTML5
- CSS3 (External)
- PHP Native
- MySQL

## Struktur Database
Database: keuangan
Tabel: transaksi (id, tanggal, keterangan, jenis, jumlah, created_at)


## Cara Menjalankan
1. Import database.sql ke phpMyAdmin
2. Taruh folder di htdocs XAMPP
3. Buka localhost/UAS_FAIQ_KEUANGAN
